Student Portfolio - Yasmin K
This is a clean, responsive, and professional digital portfolio built for academic reviews, internship applications, and personal branding.
🚀 Features
 Professional Design: Clean, modern layout using a professional blue color palette.
 Fully Responsive: Optimized for both desktop and mobile viewing.
 Section-based Organization: Clearly separated sections for personal details, resume, projects, and certifications.
 Project Modals: Interactive modals to showcase detailed project information.
 Theme Toggle: Light/Dark mode support (if implemented in JS).
🛠 Tech Stack
 HTML5: Structured semantic content.
 CSS3: Styling, grid layouts, and responsive design.
 JavaScript: Interactivity, modals, and data management.
📂 Project Structure
 ⁠index.html⁠: The main structure of the portfolio.
 ⁠style.css⁠: All styling and visual design.
 ⁠script.js⁠: Handles functionality, animations, and project data.
